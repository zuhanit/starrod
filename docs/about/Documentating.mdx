Starrod는 [MDX](https://mdxjs.com/)를 활용해 문서를 작성하고 있습니다.
이를 통해 JSX와 같은 JavaScript 언어를 직접적으로 사용하는 등
[많은 이점](https://mdxjs.com/blog/v2/)을 얻을 수 있습니다.

## 구조

- [docs](https://github.com/zuhanit/starrod/tree/main/docs)

  실제 문서가 저장되는 공간입니다.

  - .mdx

    사이트에서 볼 수 있는 문서입니다.

  - [index.yaml](https://github.com/zuhanit/starrod/blob/main/docs/index.yaml)

    문서 색인을 위한 파일입니다. list 내에는 카테고리 ID가 작성되어야하며,
    이후 문서를 지시할 경우 파일명과 동일하게 작성해야합니다.

## 문서 수정

문서 수정은 docs 폴더 내의 MDX 파일을 수정하면 됩니다.

## 문서 작성

문서를 작성하고 난 뒤에는 index.yaml을 수정해야하는 것을 잊지 마세요! 이 과정이
생략되면 화면 좌측의 사이드바에 문서가 나타나지 않게 됩니다.
