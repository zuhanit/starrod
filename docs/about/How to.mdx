Starrod에 관심이 있는 누구던지 GitHub(또는 Git)를 통해 프로젝트에 기여할 수 있습니다!
아래의 내용을 따라 기여를 시작해보세요.

## 설치

설치에 앞서, Starrod는 작동을 위해 아래 프로그램들을 필요로 합니다.

- [Git](https://git-scm.com/)
- [Node.JS](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/)

### 소스코드 다운로드

Starrod 레파지토리를 GitHub에서 클론할 수 있습니다.

```shell
git clone https://github.com/zuhanit/starrod.git
```

레파지토리를 포크하고 싶다면 아래 명령어를 입력하세요.

```shell filename="index.js"
git clone https://github.com/<<<깃허브 아이디>>>/starrod.git
```

### 빌드

의존 파일들을 설치하거나 빌드하는데에는 `Yarn`을 사용합니다.

```shell
cd starrod
yarn
```

### 실행

이제 Starrod를 빌드하거나 실행할 수 있습니다!

- `next dev` [next.js](https://nextjs.org/)를 사용해 Starrod를 개발 모드로 실행.
- `next build` Starrod 사이트 빌드.
- `next start` next.js 프로덕션 모드로 실행.
- `next lint` 린트하기.
- `storybook start-storybook -p 6006",` Starrod의
  [Storybook](https://storybook.js.org/)을 6006번 포트에서 실행.
- `build-storybook build-storybook` Starrod의 Storybook을 실행.

### 풀 리퀘스트

수정을 마쳤다면 [Pull Request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request-from-a-fork)를 생성할 수 있습니다.
